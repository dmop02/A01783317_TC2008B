Border

Border refers to the border of which-ever type of title or subtitile that a user wants to "Highlight" in the document

Border left

Border left refers to the left side of the border within the text

Border right

Border right refers to the right side of the border within the text

Border top

Border top refers to the top side of the border within the text

Border bottom

Border bottom refers to the bottom side of the border within the text

Border Radius

Border Radius refers to the radius of the border within the text

Background Image

Background Image refers to the image that is to be used as the background of the document

Background color

Background color refers to the color of the background in the text

Font

Font refers to the font that is to be used in the document or certain text in the document like the one used for this text

Font family

specifies a prioritized list of one or more font family names and/or generic family names for the selected element. For example Times and Helvetica are families

Font style

specifies a prioritized list of one or more font styles for the selected element. For example bold and italic are styles

Font weight

List of one or more font weights for the selected element. For example bold and bolder are weights

Width

Width refers to the width of the text within the document

Height

Height refers to the height of the text within the document

List style

  • List style refers to the list style of the text within the document (Circle dots, etc..)
  • List-style-type refers to the type of list style that is to be used (Emojis, Numbers, or other)

Margin

Margin refers to the margin of the text within the document. As you can see the margin for the text is different from that of the rest of the document

Padding

padding properties are used to generate space around an element's content, inside of any defined borders.

Overflow

overflow properties are used to control the way the content of an element is laid out within the document.
